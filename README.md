## 环境

```
"hexo": "^7.3.0"
```

## 快速开始

1. clone the repo to your`/theme/`folder

   ```
   git clone https://github.com/FuShaoLei/hexo-theme-white
   ```

2. change the root `_config.yml`

   ```
   theme: hexo-theme-white
   ```
   
3. done

   ```
   hexo s
   ```
